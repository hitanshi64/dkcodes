HTML

What is HTML?

  • HTML stands for Hypertext Markup Language.
  • HTML is the starndard markup language for creating webpages.

Hypertext refers to thee way in which web pages ( HTML documents) are linked together. Thus, the link available on a webpage is called Hypertext.

As its name suggests, HTML is a Markup Language which means you use HTML to simply "Mark-up" a text document with tags that tell a web browser how to structure it to display.

HTML document structure:

A typically HTML document will have the following structure:

Document declaration tag

< html>

< head>

< title> Page Title < /title>

< /head>


< body>

< h2> Heading content< /h2>

< p> paragraph content < /p>

< /body>

< /html>

Explain structure

  • The < !Document html> declaration before the < html> tag. < !Document> tag is used to inform the browser about the version of HTML used in the document.
  • The < html> tag is used to indicate the beginning and end od an HTML element in an HTML document.
  • The < head> tag is used for the head section of an HTML document.
  • The < title> tag is used for specifies a title for the HTML page.
  • The < body> is tag used for define the body of the HTML document.
  • The < h1> to < h6> tag is used for define heading content in different sizes.
  • The < p> tag is used to define paragraph content.
